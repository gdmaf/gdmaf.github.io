GDCount:
análise de grãos
com inteligência artificial.

Ferramenta de código aberto para caracterização automatizada de tamanho de grão em imagens de microscopia eletrônica de varredura (MEV). Combina rede neural convolucional U-Net com o modelo Segment Anything (SAM) para segmentação precisa sem requisitos computacionais avançados.

  • Segmentação automática por U-Net + SAM ViT-B
  • Interface acessível — sem hardware especializado
  • Distribuído como .exe (Windows) e código-fonte Python
  • Código aberto, licença MIT
